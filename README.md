## Getting Started
- clone the repo

## Installation

##### For frontend
1. Change directory
    ```sh
    cd client
    ```

2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the application
    ```sh
    npm start
    ```
#### For backend
1. Change directory
    ```sh
    cd server
    ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the application
    ```sh
    node index.js
    ```
